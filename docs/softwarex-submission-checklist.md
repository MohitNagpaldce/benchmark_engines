# SoftwareX Submission Checklist

This checklist tracks the repository and manuscript items needed before submitting this project to SoftwareX as an Original Software Publication.

## Repository

- [x] Public GitHub repository: `https://github.com/MohitNagpaldce/benchmark_engines`
- [x] Source code under `src/`
- [x] Recognized open-source license: GPL-3.0-only
- [x] README with installation, execution, metrics, and workload documentation
- [x] Citation metadata in `CITATION.cff`
- [x] Changelog for versioned release
- [x] GitHub release `v1.0.0`
- [ ] Optional DOI archive through Zenodo or another repository

## Manuscript

- [x] Local SoftwareX draft prepared from the research paper
- [ ] Move final text into the official SoftwareX Original Software Publication template
- [ ] Keep manuscript under 3000 words, excluding title/authors/affiliations/references/metadata tables
- [ ] Include no more than six figures
- [ ] Include SoftwareX metadata table with repository URL and license
- [ ] Include data availability statement
- [ ] Include declaration of generative AI use, if applicable
- [ ] Include CRediT author contribution statement
- [ ] Spell-check and grammar-check before upload
